Why AI won't kill off interaction designers

Most adults would be able to understand the code you're looking at, even if they're not a developer.

For example, here's a list:

  • HTML is for writers, to create content
  • CSS is for designers, to create pages
  • Javascript is for developers, to make the web work like a piece of software

Yes, you could go to chat GPG to write this code for you. Maybe one day humans won't write code at all.

We need to remember: behind every app or bit of software there are teams of people collaborating with each other. Web pages built out of HTML, CSS and Javascript are the end product. We only get to them after we've had conversations with each other and done research on real people.

If Interaction Designers don't need to write code, we'll move on to a difference kind of 'prototype'.
